    接口

    日期接口

    最近写一个程序的时候用到节假日和周末是否有补班，在网上找到了判断日期的。

    检查一个日期是否是节假日、工作日、休息日

    • http://www.easybots.cn/api/holiday.php?d=20240204
    • http://api.goseek.cn/Tools/holiday?date=20240204
    • http://tool.bitefu.net/jiari/?d=20240204

    检查多个日期是否是节假日、工作日、休息日

    • http://www.easybots.cn/api/holiday.php?d=20240204,202402010,20240218
    • http://tool.bitefu.net/jiari/?d=20240204,202402010,20240218

    获取一个月或多个月所有节假日

    • http://www.easybots.cn/api/holiday.php?m=202402
    • http://www.easybots.cn/api/holiday.php?m=202401,202402

    返回数据：{“201908”:{“03”:”2”,”04”:”2”,”10”:”1”,”11”:”1”,”17”:”1”,”18”:”2”,”24”:”2”,”25”:”2”,”31”:”1”}}，JSON字符串键值：日期/状态码

    状态码：工作日对应结果为 0, 休息日对应结果为 1, 节假日对应的结果为 2

    获取全年所有节假日

    • 无调休：http://tool.bitefu.net/jiari/?d=2024
    • 有调休：http://timor.tech/api/holiday/year/2024

    网页快照截图接口使用

    http(s)://s0.wp.com/mshots/v1/网址?w=宽度&h=高度

    示例

    http(s)://s0.wp.com/mshots/v1/www.zhanzhangshequ.com?w=1900&h=940
