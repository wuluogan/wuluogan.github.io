项目​

PicX​

PicX 是一款基于 GitHub API 开发的图床工具，提供图片上传托管、生成图片链接和常用图片工具箱服务。

仓库地址：https://github.com/XPoet/picx 演示地址：https://picx.xpoet.cn/#/upload

Create Chart​

基于Echarts的可视化数据大屏设计器（包含设计、预览、分享等全部流程）。

仓库地址：https://github.com/food-billboard/create-chart 演示地址：https://food-billboard.github.io/create-chart/index.html#/

takenote​

为开发人员开发的以 Web 形式存在的笔记软件。

仓库地址：https://github.com/taniarascia/takenote 演示地址：https://takenote.dev/app

PPTist​

基于 Vue3.x + TypeScript 的在线演示文稿（幻灯片）应用，还原了大部分 Office PowerPoint 常用功能，实现在线PPT的编辑、演示。支持导出PPT文件。

仓库地址：https://github.com/pipipi-pikachu/PPTist 演示地址：https://pipipi-pikachu.github.io/PPTist/

网易云音乐 API​

网易云音乐 Node.js API service。

仓库地址：https://github.com/Binaryify/NeteaseCloudMusicApi 演示地址：https://neteasecloudmusicapi-docs.4everland.app/

Win11React​

win11React 是一个开源项目，旨在使用 React、CSS 和 JavaScript 等标准 Web 技术在 Web 上复制 Windows 11 桌面体验 win11React_CN 是win11React的一个本土化项目，本项目为win11React提供近乎完整的中文翻译。

仓库地址：https://github.com/inwinter04/win11React_CN 演示地址：https://win11.iamdt.eu.org/

Soybean Admin​

一个基于Vue3、Vite3、TypeScript、NaiveUI 和 UnoCSS的清新优雅的中后台模版。

仓库地址：https://github.com/honghuangdc/soybean-admin 演示地址：https://soybean.pro/

tldraw​

A very good whiteboard.（好用的白板工具。）

仓库地址：https://github.com/tldraw/tldraw 演示地址：https://www.tldraw.com/

scrcpy​

在GNU/Linux, Windows 和 macOS上显示并控制通过 USB (或 TCP/IP) 连接的安卓设备，不需要任何 root 权限。

它专注于：

  • 轻量 (原生，仅显示设备屏幕)
  • 性能 (30~60fps)
  • 质量 (分辨率可达 1920×1080 或更高)
  • 低延迟 (35~70ms)
  • 快速启动 (最快 1 秒内即可显示第一帧)
  • 无侵入性 (不会在设备上遗留任何程序)

文章​

中国历史上最神奇的六大巧合！​

一，朱元璋年幼时叫朱重八，重八也可以理解为两个八，也就是十六天。而明朝历代皇帝的数量也刚好是十六位。

二，隋炀帝扬广为了能够登上皇帝的宝座，竞然谋害了自己的亲哥哥扬勇。让人意想不到的是，1400年从后，隋炀帝的墓穴被一个开发商意外的挖开了。而这位开发商老扳的名字就叫扬勇。

三，刘邦在芒砀山起义，被一条大蟒蛇拦住了去路，刘邦借着酒劲把这条大蟒蛇拦腰斩成了两段。从此，汉朝40O年的江山从此分为东西各2O0年，而拦腰斩断汉朝的人就叫王莽。

四，公元前206年，刘邦从秦三世子婴手中夺过江山，建立了西汉。200年以后，王莽从西汉的刘婴手中也夺过了江山。刘婴史称"孺子婴″。

五，刘备建立了蜀汉，被晋王司马昭所灭，此后凡是以晋为国号的，都灭在了刘姓的手中，刘聪灭了西晋，刘裕灭了东晋。

六，元朝开国皇帝登基的宫殿叫大明殿，万万没想到，下一个朝代就是明朝，而明朝也建立了一个乾清殿。结果，明朝灭亡之后，就有了清朝。
