# 销售

## 六大业务精髓

经过十八年的社会奋斗，从为人打工到自己创业，我总结了六大业务精髓：

**1、眼睛一红，心就黑了**

在利益的诱惑和驱使下，人们很容易失去理智，一失去理智，就会做错事。有时为了达到目的，有的人甚至不择手段，这是人性的恶。

人的眼睛和心是相连的。心里怎么想，就会从眼睛里表现出来，眼睛看到什么，心里就会有欲望。所以，贪心，眼睛红;眼睛一红，心就黑了。

你要明白，人不可怕，因为看得见;人心可怕，因为猜不透，很多人都有一副好脸，但不一定有好心肠。

**2、人越穷，越接近人性**

中国有句古话叫穷山恶水出刁民，也许不太好听，但是你不得不承认，人越穷，越容易产生人性的邪恶。由于长期生活在社会底层，他们每天都被生活的重担压得喘不过气来，没有时间和精力去思考，所以动物的本性就会暴露出来。

很多时候，为了自己的利益，他们有什么社会规则，亲戚朋友，江湖道德?都可以扔掉。我没有任何贬低穷人的意思，我自己也常常觉得穷，只是说明一个事实。

因此，与穷人打交道，直接说利益会更有效。与其直接告诉他钱的效果，不如告诉他人情、理性、未来。

**3、做自私的事，说无私的话**

嘴上说的不一定是心里想的，心里想的往往不会说出来;嘴上说的是为了大家，心里想的是为了自己;嘴上是仁义道德，底下是利益分配。这两条规则都是为个人服务的，一个人所说的任何话，都必须有自己的利益和立场。

不管怎样，不管他是有意还是无意，不管他是否意识到，他都在为自己说话，只是，有时你看到有时你看不见。每个人表面上都是彬彬有礼、恭敬的，其实有一套内在的利益机制来控制人们的各种行为。

有些事只能做而不能说，有些事只能说而不能做。走在社会上的人，千万不要被仁义道德所迷惑。

**4、隐藏的目的越深，实现的几率就越高**

曹德旺常说：“隐功能积德，为善不为人知。”意思是，有功劳要隐藏，做好事不要告诉别人，这样才能真正积德。他真的很了解这种智慧，所以他取得了不朽的成就。为什么目的隐藏得越深，实现的几率就越高?

说实话，这个世界上很少有人希望你活得好，尤其是不希望你活得比他好，当他看到你上去时，他会感到嫉妒和悲伤，会把你拉下来。所以，要么你要比他强很多，让他够不着，要么藏着掖着，不让他知道。

**5、没钱不听话，有钱赚就听话**

其实，更重要的原因是第一个，很多时候钱到位，受点委屈也是可以的。许多老板抱怨员工管理不善，说重担心他离开，说轻不起效果。其实也是因为钱不到位，钱到位，让你说他骂他，看在钱的脸上，他也会继续做下去。

不要相信那些说钱不重要的人，要么他太有钱了，钱对他来说就是一个数字，要么他太穷了，从来没有经历过有钱的幸福。无论是管理员工，还是与他人合作，记住一句话：没有钱赚谁都不听话，有钱赚谁都很听话。

**6、凡是现在让你舒服的，以后一定会让你痛苦**

人生在世，有两件事特别重要，一是抑制欲望，二是抵制诱惑，欲望是七情六欲，诱惑是财色名利。把这两件事做好，可以避免90/%以上的灾难，其实归根结底就是自制力，能够控制自己、延缓满足感的人更有可能成功。

现在能忍住的人，以后会容易得多。
