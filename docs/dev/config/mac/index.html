重装指南​

准备工作​

在开始重装系统之前,需要准备以下内容:

数据备份​

  • 使用 Time Machine 备份(推荐)
  • iCloud 云同步重要文件
  • 手动备份到外接硬盘

兼容性检查​

  • 查看设备型号
  • 检查系统要求
  • 确保有足够存储空间(建议预留40GB以上)

系统下载​

下载macOS系统:

App StoreApple官网

网络准备​

  • 确保网络稳定
  • 建议使用有线网络

💿

制作启动盘(可选)​

制作macOS启动U盘的步骤

准备U盘​

  • 准备一个至少16GB的U盘
  • 格式化为Mac OS扩展(日志式)
  • 命名为USB(便于识别)

终端命令​

使用终端命令制作启动盘:

  • 打开终端应用
  • 根据macOS版本选择命令:
  • Monterey: sudo /Applications/Install macOS Monterey.app/Contents/Resources/createinstallmedia --volume /Volumes/USB
  • Big Sur: sudo /Applications/Install macOS Big Sur.app/Contents/Resources/createinstallmedia --volume /Volumes/USB
  • Ventura: sudo /Applications/Install macOS Ventura.app/Contents/Resources/createinstallmedia --volume /Volumes/USB

注意事项​

  • 制作过程中不要拔出U盘
  • 需要输入管理员密码
  • 等待进度条完成

🔄

进入恢复模式​

进入macOS恢复模式的方法

Intel芯片Mac​

  • 关机后按住Command(⌘) + R键开机
  • 等待出现苹果标志和进度条

M1/M2芯片Mac​

  • 关机后长按电源键
  • 等待出现启动选项
  • 选择选项图标

其他恢复模式组合键​

  • Command + Option + R: 在线恢复最新兼容的macOS
  • Shift + Command + Option + R: 恢复出厂预装的macOS

继续操作​

  • 选择语言后点击继续

🗑️

磁盘工具操作​

使用磁盘工具准备安装

打开磁盘工具​

  • 在恢复模式中选择磁盘工具
  • 在左侧边栏选择内置硬盘

抹掉磁盘​

  • 点击抹掉按钮
  • 名称: Macintosh HD
  • 格式: APFS (加密)
  • 方案: GUID分区图

分区建议​

  • 单分区(推荐)
  • 如需双系统,请提前规划分区

完成操作​

  • 完成后退出磁盘工具

⚙️

安装系统​

开始安装macOS系统

选择安装选项​

  • 选择重新安装macOS
  • 点击继续
  • 同意许可协议
  • 选择安装位置(Macintosh HD)

安装过程​

  • 下载系统文件(需要网络)
  • 验证下载
  • 准备安装
  • 自动重启数次

注意事项​

  • 安装过程中请勿关机
  • 确保电源连接
  • 保持网络通畅

安装时间​

  • 预计耗时:30-60分钟

⚡

初始设置​

完成系统初始设置

基础设置​

  • 选择语言和地区
  • 连接WiFi网络
  • 登录Apple ID
  • 设置用户账户

迁移助理​

  • 从Time Machine备份恢复
  • 从其他Mac迁移
  • 从Windows PC迁移
  • 不传输数据(全新开始)

系统设置​

  • 启用/禁用Siri
  • 配置触控板
  • 设置屏幕时间
  • 开启查找我的Mac
