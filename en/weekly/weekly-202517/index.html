开源​

chatlog​

微信聊天记录工具，帮助大家轻松使用自己的聊天数据：

  • 从本地数据库文件获取聊天数据
  • 支持 Windows / macOS 系统
  • 支持微信 3.x / 4.0 版本
  • 提供 Terminal UI 界面 & 命令行工具
  • 提供 HTTP API 服务，支持查询聊天记录、联系人、群聊、最近会话等信息
  • 支持 MCP SSE 协议，可与支持 MCP 的 AI 助手无缝集成
  • 支持多媒体消息，支持解密图片、语音
  • 支持自动解密数据，简化使用流程
  • 支持多账号管理，可在不同账号间切换

网站​

CHAZ 3D – Nokia 3250​

CHAZ 3D – Nokia 3250是一个在线Nokia 3250 3D体验网站，该3D体验展示诺基亚3250经典手机的交互式模型，支持拖拽旋转、捏合缩放以及长按电源键实现开关机操作。用户可在操作区域内通过屏幕触控感受手机基本功能。

VmShell​

VmShell 提供永久免费的公共 MySQL 数据库服务，数据库位于 mysql.vmshell.com，端口号为 3306，用户可以使用注册时设置的用户名和密码进行连接

注册地址：https://mysql.vmshell.com/register.php

数据库地址数据库地址：mysql.vmshell.com 端口号：3306
